## Overview
AI interviewer is a interview practivce platform that enables users to practice for job interviews so they can land their dream job sooner 

## Features
1. Secure authentication and profile management
2. Personalized interview practice 
3. intelligent resume parsing 
4. constructive feedback and useful insights after interview
5. more to come..

## Tech stack 
1. Client application: React JS , Ant design , Tailwind CSS
2. Server application: FastAPI , UV 
3. Database: Supabase 
4. Intelligence layer: Gemini API , Langchain
